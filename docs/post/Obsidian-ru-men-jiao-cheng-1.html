Obsidian入门教程1

入门篇：为什么是Obsidian？

大家好呀 👋

不知道你有没有这样的经历：

  • 看到一篇好文章，保存到微信收藏夹，b站收藏夹、小红书收藏里，然后就嫌麻烦再也没打开过
  • 学习了一个新知识，过几天就忘了，想要复习却不记得埋藏在哪里
  • 笔记记了一大堆，但散落在zotero、wps里，找都找不到
  • 学了很多东西，但知识都是孤立的，串不起来

如果有的话，那你可能跟我一样——需要一个真正的知识库，而不仅仅是"记笔记"。

今天这篇教程，我想带你认识一个我用了很久的工具——Obsidian，并带你从零开始搭建属于你自己的知识库。

一、什么是Obsidian？

Obsidian 是一款本地优先的笔记软件，听起来很普通对吧？但它有几个非常特别的地方：

  1. 你的数据永远是你自己的 —— 所有笔记都是纯文本 Markdown 文件，存在你的电脑本地，就算软件倒闭了，你的笔记一个都不会丢
  2. 双向链接 —— 这是 Obsidian 最核心的魅力，笔记之间可以互相链接
  3. 图谱视图 —— 你的笔记连接起来后，会形成一个网络图，一眼就能看到你的知识结构
  4. 插件系统 —— 功能不够？社区有上千个插件，几乎可以满足你所有需求

Obsidian 不是一个"记笔记"的工具，而是一个帮你"思考"和"连接"的工具。

二、下载与安装

第一步很简单，去官网下载即可：

👉 https://obsidian.md

支持 Windows / macOS / Linux / iOS / Android，全平台覆盖。

下载安装后打开，你会看到这个界面：

三、创建你的第一个仓库（Vault）

Obsidian 里，一个"仓库"（Vault）就是一个文件夹，里面装着你的所有笔记。

点击 "创建新仓库"，给它起个名字，比如：

我的知识库

然后选择一个存放位置，比如桌面或者你的文档文件夹。

📁 打开这个文件夹你会发现——里面什么都没有！就是一个普通的文件夹。因为 Obsidian 的笔记就是普通的 .md 文件，你可以用任何文本编辑器打开它们。

四、认识Obsidian界面

创建好仓库后，你会看到 Obsidian 的界面分为几个区域：

  • 左侧栏：文件浏览器、搜索、切换等功能
  • 中间：你写笔记的地方，也是整个软件的核心
  • 右侧栏：查看反向链接、图谱等辅助功能

💡 刚上手不需要记太多，你只需要记住：中间写笔记，左边找文件，就够了。

五、创建你的第一篇笔记

按下 Ctrl + N（Mac 是 Cmd + N），创建一个新笔记。

输入以下内容试试：

# 我的第一篇笔记

你好，Obsidian！这是我的第一个知识库笔记。


今天学到了：


    

  • Obsidian 是本地优先的笔记软件
    • 

  • 笔记文件是 Markdown 格式
    • 

  • 一个 Vault 就是一个文件夹

六、Markdown——知识库的基石

Obsidian 使用 Markdown 语法，非常简单，几分钟就能学会。

常用的语法：

效果 语法
粗体 粗体
斜体 斜体
标题 # 一级 ## 二级 ### 三级
列表 - 项目 或 1. 项目
链接 文字
代码 代码
引用 > 引用内容

最重要的是：Obsidian 培养的是一种链接式思维——你不再孤立地记录知识，而是主动去建立联系。这本身就是一种非常高效的学习方式 🧠

以上就是Obsidian入门的第一篇内容。通过本文，你已经学会了如何安装、创建仓库、写第一篇笔记，并体验了双向链接的魅力。下一期教程中，我们将深入讲解如何自定义常用快捷键、建立标签、文件夹和模板来构建你的知识库结构，让知识管理更高效。现在就可以动手尝试，创建你的第一篇带链接的笔记吧！

七、Obsidian 的灵魂：双向链接 ✨

这是 Obsidian 区别于其他笔记软件最核心的功能。

创建链接很简单，只需要用两对方括号：

今天学习了 [[双向链接]] 的用法。

这样，双向链接 就变成了一个链接，点击它就会跳转到（或创建）同名的笔记。

为什么这很重要？

想象一下，你在学一个知识：

  • 你在"Python"笔记里，提到了"数据分析"
  • 你在"数据分析"笔记里，提到了"可视化"
  • 你在"可视化"笔记里，提到了"Matplotlib"

如果没有链接，这三篇笔记是孤立的。有了链接，它们就变成了一个知识网络。你可以轻松地从 Python 一路导航到 Matplotlib，就像在维基百科里跳转一样。

反向链接更是神器——在任何一篇笔记的底部，Obsidian 会自动显示"有哪些笔记提到了这篇笔记"。

举个例子：如果你在 10 篇不同的笔记里提到了"项目管理"，那么打开"项目管理"笔记时，底部会列出所有引用它的笔记。这就帮你发现了那些你都没意识到的知识关联。

八、看一眼你的知识图谱 🕸️

创建了几篇带链接的笔记后，点击左侧的 图谱视图 图标（一个圆点+连线），你会看到：

每一个圆点是一篇笔记，每一条线是一个链接。这就是你的知识网络

刚开始可能只有几个点，但随着笔记越来越多，这个图会变得越来越丰富。

九、选择一个喜欢的主题进行设置

Obsidian 默认使用的是浅色主题，但如果你喜欢夜间写笔记，或者想要更美观的界面，可以换个主题。

点击左下角的 齿轮图标 ⚙️ → 外观，你会看到：

  • 主题：内置了浅色和深色两种主题，可以随时切换
  • 社区主题：点击"管理"可以浏览社区主题，有上千款供你选择
  • 字体：可以自定义界面字体和编辑区字体

推荐几个热门的社区主题（安装后即可在主题列表切换）：

主题名称 特点
Minimal 简约干净，性能极佳，生态最丰富
Blue Topaz 颜值超高，适合中文用户
Things 灵感来自 Things 应用，清爽精致
AnuPpuccin 色彩柔和，高度可定制
这是我个人常用的主题，觉得比较好看舒服，推荐一下

💡 主题换了之后，整个界面的气质完全不同。建议选一个看着舒服的，不用频繁更换——内容才是核心。
